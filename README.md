# SOC Incident Reports

## Overview
This repository contains documented Security Operations Center (SOC) investigations completed in a simulated SIEM environment using the TryHackMe SOC Simulator.

The purpose of this repository is to demonstrate SOC Level 1 analyst skills, including alert triage, log analysis, investigation methodology, and clear incident documentation.

---

## Environment
- Platform: TryHackMe
- Simulation: SOC Simulator
- Role Focus: SOC Level 1 / Junior Security Analyst
- Data Type: Simulated (no real organizational data)

---

## Repository Structure
Each incident report represents a single SOC alert and includes:
- Alert description and severity
- Investigation steps and evidence reviewed
- Findings and analyst reasoning
- Final classification (True Positive / False Positive)
- Action taken and conclusion

---

## Incident Coverage
The incidents in this repository cover multiple SOC alert categories:
- Endpoint execution activity
- Authentication and identity anomalies
- Network-based indicators of compromise

---

## Skills Demonstrated
- SIEM alert triage
- Log correlation and contextual analysis
- False-positive identification
- Incident classification and closure
- SOC-standard written reporting

---

## Disclaimer
All investigations are based on simulated lab environments and are intended solely for educational and portfolio demonstration purposes.
